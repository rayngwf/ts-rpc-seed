> a typescript node rpc seed project.

## Build Setup

``` bash
# install dependencies
cnpm i

# compile in watch mode
npm run build

# debug with ts-node directly
vscode -> F5 -> tsnode

# test with a quick and dirty client
node dist/client.js
```
